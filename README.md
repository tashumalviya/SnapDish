# SnapDish - Share Every Bites

A modern and responsive recipe application built with **React.js, Vite, and Tailwind CSS**. SnapDish helps users explore recipes with a clean UI and smooth user experience.

## Live Demo -- Vercel

🔗 https://snap-dish-two.vercel.app/

## Features

*  Browse and explore recipes
*  Easy recipe discovery
*  Fully responsive design
*  Modern and clean UI
*  Fast performance with Vite
*  Component-based React architecture

##  Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Language:** JavaScript
* **Deployment:** Vercel

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/tashumalviya/SnapDish.git
```

Go to project directory:

```bash
cd SnapDish
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173/
```
