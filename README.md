# SnapDish (Plain JSX + CSS Export)

Standalone export of the SnapDish recipe app — pure React + Vite + Tailwind CSS v4. No TypeScript, no TanStack Start.

## Install & run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Stack
- React 18 (JSX only)
- Vite 5
- Tailwind CSS v4 (via `@tailwindcss/vite`)
- React Router DOM v6
- Framer Motion, Lucide icons, Sonner toasts

All styling lives in `src/styles.css` (design tokens + utilities).
